05相机去畸变

发表于2026-03-02|更新于2026-03-03|传统视觉
|浏览量:

总览

第一部分完成相机标定流程，流程覆盖图像预处理、棋盘格角点检测、多尺度策略、亚像素级精确化、标定求解、重投影误差评估、参数保存与去畸变验证。第二部分实现手写去畸变流程，流程使用标定得到的内参和畸变系数，在像素级进行畸变模型反算，并用最近邻插值生成去畸变图像。

第一部分：标定过程（ROS + OpenCV）

1. 程序入口与参数来源

程序以 ROS 节点形式运行，节点名为 camera_calibration_node。程序从 ROS 参数服务器读取核心参数，并提供默认值：

  • image_dir：标定图像目录
  • board_width / board_height：棋盘格内角点数量（列数/行数）
  • square_size：棋盘格单元边长（米）

程序使用 boost::filesystem 检查目录存在性并遍历目录文件，然后按文件名排序，保证处理顺序稳定。

2. 世界坐标系角点构建

程序预先构造棋盘格在世界坐标系下的三维点 object_points。棋盘格被放置在 Z=0 平面上，点按行列展开，单位为 square_size：

  • X = j * square_size
  • Y = i * square_size
  • Z = 0

程序将该模板在每张成功检测的图像上重复使用，并加入 object_points_vec。

3. 图像预处理与低对比度增强

程序对输入图像进行统一预处理，处理函数 handleLowContrast() 走通用路径：

  1. 将彩色图转换为灰度图
  2. 使用 CLAHE 进行自适应直方图均衡化
  3. 输出增强后的灰度图用于角点检测

CLAHE 的裁剪限制 clipLimit 设置为 2.0，该设置提高局部对比度并抑制过度增强的噪声。

4. 多尺度检测策略

程序根据图像最大边长对输入图进行缩放，控制角点检测在合适的尺度上运行：

  • 当图像过大时执行降采样，降低计算开销
  • 当图像过小时执行上采样，提高角点显著性
  • 当图像处于合理区间时保持原始尺度

缩放因子 scale_factor 被记录，用于后续将检测到的角点坐标映射回原始分辨率。

5. 棋盘格角点检测的多策略组合

程序使用 cv::findChessboardCorners() 执行角点检测，并组合不同标志位以提升鲁棒性：

  • CALIB_CB_ADAPTIVE_THRESH：自适应阈值增强边缘对比
  • CALIB_CB_FAST_CHECK：快速预检查提升速度
  • CALIB_CB_NORMALIZE_IMAGE：归一化输入图像改善检测稳定性

当第一次检测失败时，程序执行备用策略：

  1. 对图像做 GaussianBlur 降噪与平滑
  2. 使用较少标志位再次尝试
  3. 最后尝试更简化的检测标志组合

该多策略路径提升了弱纹理、曝光不均、噪声更强场景下的成功率。

6. 亚像素级角点精确化与坐标回映射

检测到角点后，程序使用 cv::cornerSubPix() 提升角点精度。程序使用较严格的终止条件与更大的搜索窗口：

  • 终止条件：最多 50 次迭代或误差小于 0.001
  • 搜索窗口：(15,15) 与最终 (21,21) 两阶段细化

当检测图像执行过缩放时，程序将角点坐标按 1/scale_factor 映射回原始图像坐标，再在原始尺度上进行最终亚像素优化。

7. 标定求解与畸变模型约束

程序调用 cv::calibrateCamera() 求解相机内参和畸变参数。程序明确将畸变系数矩阵设置为 4 维，只保留 k1, k2, p1, p2，并通过标定 flags 固定高阶畸变项：

  • 固定 k3
  • 固定 k4
  • 固定 k5
  • 使用 LU 分解 CALIB_USE_LU

该约束适用于多数普通镜头在畸变不极端时的工程实践，有助于稳定求解并减少过拟合风险。

8. 重投影误差评估与坏样本标记

程序对每张成功图像计算重投影误差：

  1. 使用 cv::projectPoints() 将三维角点投影回图像
  2. 计算检测角点与投影角点的 L2 误差
  3. 归一化为每个视图的 RMS 误差并存储

程序将误差超过 2.0 像素的图像标记为高误差样本，以便后续剔除或重新采集。

9. 参数输出、保存与验证

程序打印并保存如下结果：

  • 图像尺寸
  • RMS 误差与总体重投影误差
  • 内参 fx, fy, cx, cy
  • 畸变 k1, k2, p1, p2（k3 固定为 0）

程序将结果写入 calibration_results.txt，并使用第一张图像做一次 cv::undistort() 去畸变对比，左右拼接后显示用于快速肉眼验证。

第二部分：去畸变过程

1. 参数输入与目标

程序使用已知标定参数进行去畸变，不依赖 OpenCV 的 undistort，而是直接按畸变模型计算像素映射。程序使用的畸变模型为常见的径向畸变 + 切向畸变：

  • 径向畸变：k1, k2
  • 切向畸变：p1, p2

程序使用内参 fx, fy, cx, cy 完成像素与归一化坐标系的转换。

2. 像素级映射计算流程

程序对输出图像的每个像素 (u, v) 执行如下步骤：

  1. 将输出像素转换为归一化坐标：
  • x = (u - cx) / fx
  • y = (v - cy) / fy
  1. 计算半径：
  • r = sqrt(x^2 + y^2)

  1. 计算畸变后的归一化坐标

  2. 将畸变坐标映射回像素坐标：

  • u_d = fx * x_d + cx
  • v_d = fy * y_d + cy
  1. 使用最近邻插值从原图取值并填充到去畸变图像中：
  • 当 (u_d, v_d) 在图像范围内时取最近邻像素值
  • 当映射越界时填充 0

该实现属于逆向采样路径，输出图像像素通过查表方式从输入畸变图读取，避免输出出现空洞。
