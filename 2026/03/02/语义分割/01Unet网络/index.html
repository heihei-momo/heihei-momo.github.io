01Unet语义分割网络

发表于2026-03-02|更新于2026-03-03|语义分割
|浏览量:

U-Net 最初在2015年针对生物医学图像分割任务提出，但因其精巧的结构和出色的效果，迅速成为了语义分割领域一个经典的、里程碑式的模型，被广泛应用于医疗影像、卫星图像分析、工业检测等各个领域。

网络结构：U 形编码器-解码器

U-Net 的整体结构类似字母 “U”，由两部分组成：

  • 左侧编码器（Contracting Path）：逐步下采样，提取高层语义
  • 右侧解码器（Expanding Path）：逐步上采样，恢复空间分辨率
  • 中间瓶颈（Bottleneck）：最深层语义特征

编码器（下采样）

编码器通常由重复的模块组成：

  • Conv(3x3) + ReLU（常见是两次卷积）
  • MaxPool(2x2) 下采样

下采样使特征图尺寸变小，但通道数变多，语义信息更强。

解码器（上采样）

解码器通常由重复的模块组成：

  • 上采样（UpConv 或 Upsample + Conv）
  • 与编码器对应层的特征图进行拼接（skip connection）
  • Conv(3x3) + ReLU（常见是两次卷积）

上采样逐步恢复空间分辨率，同时融合浅层细节信息。

跳跃连接（Skip Connection）

U-Net 的关键是编码器到解码器的跳跃连接，通常做法是：

  • 将编码器某一层的特征图，直接与解码器对应层拼接（concat）

跳跃连接的作用包括：

  • 保留边缘、纹理等细节信息
  • 缓解下采样导致的空间信息丢失
  • 让解码器既拥有高层语义，也拥有低层细节

输出层与预测方式

U-Net 最后一层通常使用 1x1 Conv 将通道数映射到类别数：

  • 二分类：输出 1 个通道 + Sigmoid
  • 多分类：输出 C 个通道 + Softmax

输出为每个像素的类别概率分布。

常用损失函数

分割任务常见损失包括：

1) Cross Entropy Loss

适用于多分类分割，优化像素分类准确率。

2) Dice Loss

用于处理类别不平衡问题，尤其适合医学分割（前景区域很小）。

Dice 系数形式：

  • 前景越重叠，Dice 越大
  • Dice Loss = 1 - Dice

3) BCE + Dice

二分类常见组合，兼顾像素级与区域级目标。

4) Focal Loss

用于极端类别不平衡，增强对难样本的关注。
