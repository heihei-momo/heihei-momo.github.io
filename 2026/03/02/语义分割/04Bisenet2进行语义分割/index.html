BiSeNetV2 语义分割简介

BiSeNetV2 是面向实时语义分割设计的网络结构。语义分割需要同时做到两件事：在像素级输出每个像素的类别（语义正确），并尽可能保留边界细节（轮廓清晰）。BiSeNetV2 用“双分支”方式将这两类需求拆开处理，再进行高效融合，从而在速度与效果之间取得平衡。

语义分割任务输出

BiSeNetV2 的输出与典型语义分割一致：

  • 输入：一张图像 H x W x 3
  • 输出：与输入同尺寸的类别概率图 H x W x C
    • 多分类使用 Softmax
    • 二分类可用 Sigmoid

最终得到像素级 mask（每个像素一个类别 id）。

核心结构：双分支（Bilateral）

BiSeNetV2 的关键是两条并行分支：

1) Detail Branch（细节分支）

细节分支专门负责高分辨率空间信息：

  • 下采样较少，保持较高特征分辨率
  • 结构相对浅，计算开销可控
  • 输出包含边缘、纹理、细线条等信息

细节分支的作用是让最终分割结果在边界处更清晰，减少“糊边”和“小结构被吞掉”的问题。

2) Semantic Branch（语义分支）

语义分支专门负责高层语义理解与大感受野：

  • 下采样更强，特征图更小，计算更快
  • 通过轻量模块获取更强的语义表达
  • 常带有上下文增强，让模型理解全局场景

语义分支的作用是让分类更准确，减少“把路面分成建筑、把墙面分成路”的错误。

融合模块：BGA（Bilateral Guided Aggregation）

双分支的输出需要融合成最终用于分类的特征。BiSeNetV2 使用 BGA 模块做融合，融合逻辑强调“相互引导”：

  • 使用语义分支的高层信息，指导细节分支的有效区域
  • 使用细节分支的空间结构，补充语义分支上采样后的细节缺失

BGA 的效果表现为：

  • 边界更清晰
  • 小目标与细长结构更容易保留下来
  • 在不显著增加计算量的前提下提升分割质量

分割头（Segmentation Head）与辅助监督

主分割头

融合后的特征进入分割头，输出每像素类别概率。

辅助监督（训练阶段常用）

训练时常在语义分支的中间层加辅助输出（aux head），用于：

  • 加强语义分支学习
  • 加速收敛
  • 提升稳定性

推理部署时通常移除辅助头，不影响主输出与速度。

训练与损失要点

BiSeNetV2 常用训练配置符合语义分割通用套路：

  • 损失函数：Cross Entropy 为主；类别不平衡时可叠加 Dice/Focal 或使用 class weight
  • 数据增强：随机缩放 + 随机裁剪（对语义分割很关键），翻转、颜色抖动等
  • 输入分辨率：实时系统通常控制输入尺寸以换取速度；高精度需求则提升输入尺寸或做滑窗推理
