1. Fast-LIO2 概述

Fast-LIO2 是一种高效的激光-惯性里程计（LIO）方法，强调 紧耦合 与 实时性。它将 IMU 的高频运动信息与 LiDAR 的几何约束结合，通过 迭代卡尔曼滤波（Iterated Kalman Filter, IKF） 在每帧（或每个扫描）内多次线性化与更新，使得在快速运动、点云稠密、场景结构复杂时仍能保持稳定和高精度。

Fast-LIO2 的典型特征包括：

  • 以 滤波框架 为主线（而非传统“前端+后端图优化”）
  • LiDAR 约束使用 点到平面（point-to-plane） 残差（ICP 风格）
  • 使用 局部地图（local map） 管理与高效近邻搜索（常见为 KD-tree）
  • 重点解决 LiDAR 扫描运动导致的 点云去畸变（deskew）
  • 在更新中对非线性测量采用 迭代更新 提升收敛与鲁棒性

2. 系统输入与输出

2.1 输入

  • IMU：角速度 ω、线加速度 a（高频）
  • LiDAR：点云（较低频，但每帧点数多）
  • 可选：外参初值（LiDAR-IMU 相对位姿）、时间偏差等

2.2 输出

  • 里程计：位姿 T_wb 或 T_wl（世界到IMU/雷达）
  • 速度、IMU bias
  • 可选：局部地图、去畸变点云、轨迹发布等

3. 核心问题与解决思路

Fast-LIO2 主要解决以下工程痛点：

  1. 高速运动导致点云扭曲
    LiDAR 一帧扫描跨越一个时间窗口，机器人在扫描期间运动会造成点云“拉伸/扭曲”。
    Fast-LIO2 使用 IMU 在扫描时间内插值姿态与位置，对每个点进行时间对齐，实现去畸变。

  2. LiDAR-IMU 紧耦合估计
    使用滤波器将 IMU 预测与 LiDAR 几何测量统一在同一状态量中更新，避免松耦合带来的信息损失。

  3. 非线性测量的线性化误差
    点到平面残差是强非线性的，Fast-LIO2 通过 IKF 在同一帧内多次迭代更新，缓解一次线性化不准确的问题。

  4. 实时性与大点云处理
    局部地图与高效近邻检索减少计算负担，点云下采样与特征筛选进一步保证实时性能。

4. 状态量与误差模型

Fast-LIO2 常用的状态（以 IMU 为主体坐标系 b）可写为：

  • 姿态：R_wb（或四元数 q_wb）
  • 位置：p_wb
  • 速度：v_wb
  • IMU 偏置：b_g（陀螺 bias），b_a（加计 bias）
  • 重力向量：g_w（有些实现将重力方向/大小作为状态或常量）
  • 外参：T_bl（IMU 到 LiDAR 的位姿），部分实现支持在线标定（可选）

滤波一般在误差状态上工作（ESKF 思路）：

  • 小角度误差 δθ
  • 位置误差 δp
  • 速度误差 δv
  • 偏置误差 δb_g, δb_a
  • （可选）外参误差

5. IMU 预测（Propagation）

IMU 预测在两个 LiDAR 帧之间以高频积分：

  1. 去偏置：
  • ω_hat = ω - b_g
  • a_hat = a - b_a
  1. 姿态更新：
  • R_{k+1} = R_k * Exp(ω_hat * Δt)
  1. 加速度转到世界系并减重力：
  • a_w = R_k * a_hat - g_w
  1. 速度与位置更新：
  • v_{k+1} = v_k + a_w * Δt
  • p_{k+1} = p_k + v_k * Δt + 0.5 * a_w * Δt^2

同时传播协方差 P（使用线性化的系统矩阵 F 和噪声输入矩阵 G）。

6. 点云去畸变（Deskew）

LiDAR 每个点都有采样时间戳（或可由点的相对时间推算）。去畸变目标是把点统一变换到同一参考时刻（例如帧起始或帧结束）：

  1. 在扫描时间窗口内利用 IMU 预测获得连续姿态/位置（通常可插值）
  2. 对每个点 p_l(t_i)，用该点时刻的传感器位姿 T_wl(t_i) 将其映射到统一参考时刻坐标系
  3. 输出去畸变点云用于后续匹配

去畸变是 Fast-LIO2 能在高速运动下稳定工作的关键环节之一。

7. LiDAR 测量模型：点到平面残差

Fast-LIO2 的 LiDAR 约束通常来自局部地图中的平面结构：

  1. 从去畸变点云中取点 p（在 LiDAR/IMU 坐标系下）
  2. 根据当前状态估计将点变换到世界系：
    P = R_wb * (R_bl * p + t_bl) + p_wb
  3. 在局部地图中搜索 P 的近邻点集
  4. 用近邻点拟合局部平面（法向量 n 与平面上一点 P0）
  5. 构造残差（点到平面距离）：
    r = n^T (P - P0)

对每个有效点都能得到一个残差，堆叠后形成测量方程。

8. 迭代卡尔曼滤波（IKF）更新

点到平面残差是非线性的，Fast-LIO2 在同一帧内执行多次迭代：

  1. 使用当前状态估计进行一次匹配与平面拟合，构建残差 r(x)
  2. 在当前线性化点计算雅可比 H = ∂r/∂x
  3. 使用卡尔曼更新得到增量 δx
  4. 用 δx 更新状态（姿态用 Exp(δθ) 方式更新）
  5. 重复 1~4，直到：
    • 迭代次数达到上限，或
    • 增量足够小 / 残差收敛

迭代更新的效果是：

  • 更接近高斯-牛顿风格的迭代最小二乘
  • 对强非线性问题（尤其初值不够好时）更稳
  • 提升点云匹配收敛域与最终精度
